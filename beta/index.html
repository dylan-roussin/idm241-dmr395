Triggers

This interaction uses a hover and a click.

Rules

- On hover: The subscribe button is initally a red outline with red text. On hover, the cursor changes to a pointer, the text color changes to white and the button fills in red from the bottom.


- Off-hover: The red fill decends until the button returns to an outline. The text changes from white to red.


- On click: The subscribe button presses inward and changes color to grey. When click is released, the button pops back out.


- Click to off state: The subscribe button presses inward and changes color to red. When click is released, the button pops back out.
