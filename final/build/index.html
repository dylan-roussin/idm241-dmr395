SUBSCRIBE
link to YouTube video

Triggers

This interaction uses a hover and clicks.

Rules

- On hover: The subscribe button is initally a red outline with red text. On hover, the cursor changes to a pointer for both the button and bell, the text color changes to white and the button fills in red from the bottom.


- Off-hover: The red fill decends until the button returns to an outline. The text changes from white to red.


- On click: The subscribe button presses inward and changes color to grey. When click is released, the button pops back out. The inner text changes to "SUBSCRIBED". The bell grows to size next to the subscribe button. When clicked, the bell changes color from a light grey to a darker shade and plays a ringing animation.


- Click to off state: The subscribe button presses inward and changes color to red. When click is released, the button pops back out. The inner text changes to "SUBSCRIBE". The bell disappears. When clicked to off, the bell changes from black back to a light grey.
