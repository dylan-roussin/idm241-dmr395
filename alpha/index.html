Triggers

This interaction uses both a hover and click.

Rules

On hover, a grey box appears behind the arrow, and the arrow's border changes color to orange. When clicked, the arrow fills in orange through an animation, the number of upvotes increases by one and also turns orange. The upvote is then stored in a database.
